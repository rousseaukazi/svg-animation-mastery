Mouse Spotlight

CSS mask-image with a radial gradient follows your cursor, revealing a hidden constellation scene beneath a solid overlay.

Move your cursor (or touch) to reveal the scene

How it works

A full-screen div overlay covers a hidden SVG scene. The overlay uses CSS mask-image with a radial gradient that is transparent at the cursor position and opaque everywhere else — punching a circular "window" that follows the mouse with zero JavaScript lag on the paint side.

The cursor position is tracked via mousemove and written to CSS custom properties (--mx, --my) inside a requestAnimationFrame callback to avoid forced reflows. Touch events are also supported, making it work on mobile.

1

Layer two elements — The hidden scene SVG sits at position absolute. An overlay div sits on top. The scene is only visible where the overlay is transparent.

2

CSS mask-image radial gradient — mask-image: radial-gradient(circle 180px at var(--mx) var(--my), transparent 0%, transparent 40%, black 80%) creates the circular cutout.

3

Custom property update — overlay.style.setProperty('--mx', x + 'px') inside rAF is the most performant way to track a cursor position.

4

Touch support — Mirroring mouse events with touchmove using e.touches[0] makes the effect mobile-friendly with no extra library.

⚡ Path Drawing

Circuit Network

SVG paths reveal themselves by animating stroke-dashoffset from full path length to zero. Staggered with GSAP, glowing with SVG filters.

How it works

The trick is deceptively simple: set stroke-dasharray to the path's total length (measured via getTotalLength()), then set stroke-dashoffset to the same value. This makes the dash pattern push all strokes off-screen. Animating dashoffset to 0 "draws" the path.

GSAP staggers each path sequentially so circuits appear to spread outward from the source. An SVG feGaussianBlur filter with feMerge composites a soft glow halo beneath each stroke without any DOM overhead.

1

Measure path lengths — path.getTotalLength() returns the exact pixel length of any SVG path, including curves and arcs.

2

Set dasharray = dashoffset = length — The dash is exactly as long as the path, but offset by the same amount, making the entire stroke invisible.

3

Tween dashoffset to 0 — GSAP animates the offset to zero, sliding the visible dash onto the path. stagger: 0.08 cascades each path 80ms apart.

4

Nodes pop in with back.out easing — Junction circles scale from 0 to 1 using ease: 'back.out(2)' for a satisfying snap, then pulse forever.
